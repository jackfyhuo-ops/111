天堂風放置RPG GitHub Pages 版

使用方式：
1. 到 GitHub 建立一個新的 repository
2. 上傳這個資料夾內的 index.html
3. 到 Settings → Pages
4. Source 選 Deploy from a branch
5. Branch 選 main / root
6. 儲存後等待 1~3 分鐘
7. GitHub 會產生網址，例如：
   https://你的帳號.github.io/你的專案名稱/

iPhone 使用方式：
1. 用 Safari 開啟 GitHub Pages 網址
2. 點分享
3. 選「加入主畫面」
4. 之後就能像 App 一樣開啟

注意：
- 這是純 HTML / CSS / JavaScript 單檔遊戲
- 存檔使用瀏覽器 LocalStorage
- 換手機或清除 Safari 網站資料會清掉存檔
